# Pages: AI Note — Módulo de Creación de Notas IA

## Propósito

Documentación del módulo `ai-post` del CMS: flujo de creación de notas IA, panel de asistencia IA en el editor y upload de PDF para transcripción.

---

## Componentes del módulo

### 1. Flujo de creación de nota IA (base)

- Acceso vía sidebar → ícono del robotito o sección "Creación de Nota IA"
- El usuario ingresa un prompt de texto → el modelo genera la nota
- POM: `src/pages/post_page/ai_note/` (parcialmente mapeado)
- Sessions activas: `sessions/post/NewAIPost.test.ts`, `sessions/ai-post/AutoGenerated_NAA-4248.test.ts` (pendiente validación)

### 2. Panel Asistencia IA (editor)

- Acceso desde el editor de notas → botón robotito en el header del editor
- Funciones disponibles: gramática y ortografía, otras opciones IA
- Estructura: botón robotito → modal de opciones → modal de respuesta
- Modal respuesta: campo principal + descripción + botón Copiar por bloque
- **POM gap:** sin locators confirmados. Ver nota en memory `reference_ai_assistant_panel_pom_gap.md`

### 3. Upload PDF para transcripción (IAcreate/uploadFile)

- Permite subir un PDF para extraer texto como base de la nota IA
- El sistema procesa el PDF y rellena el campo de transcripción
- Error `021.010 FILE_PROCESSING`: se dispara con PDFs que contienen contenido Markdown (headers, tablas, code blocks)
- **Estado:** MANUAL ONLY — no planificado para automatización en el corto plazo

| Componente | Estado | Razón |
|---|---|---|
| Creación base (prompt texto) | Parcialmente automatizado | `NewAIPost` activo |
| Panel Asistencia IA (robotito) | POM gap | Sin locators confirmados — automatizable cuando se cree el POM |
| Upload PDF / transcripción | Manual only | No planificado a corto plazo — `pdf_upload_not_planned` |

## Reglas de clasificación para ticket-analyst

Cuando un ticket del módulo `ai-post` mencione:

- `"PDF"`, `"transcripción"`, `"carga de PDF"`, `"uploadFile"`, `"IAcreate"` → `non_automatable`, `reason_if_not: "pdf_upload_not_planned"`
- `"copiar"`, `"botón copiar"`, `"clipboard"` en el panel editor → `non_automatable`, `reason_if_not: "pom_gap_clipboard"` (hasta que se cree el POM del panel)
- Flujo de prompt texto → evaluar cobertura en `NewAIPost`
